## 六、生产力：HF Papers + Upwork 闭环

### 核心原则：需求先行，不是看到论文酷就做

```
❌ 看论文觉得酷 → 花时间复现 → 发现没人买单
✅ 先扫 Upwork 需求 → 带着需求看论文 → 做 demo → 投标 + 发帖
```

### 每周一次（30 分钟）

1. 打开 [HF Weekly Papers](https://huggingface.co/papers) → 只看前 10-15 篇
2. 扫标题 + 摘要（5 分钟），标记和你领域相关的
3. 交叉验证：Upwork 搜相关关键词 → 确认有人付钱
4. 选 1 篇深入，读摘要 + 方法
5. 写 LinkedIn 帖子 + 投 Upwork 标

### 你的领域过滤词

只看包含这些关键词的论文，其他跳过：
`Agent` `RAG` `Retrieval` `Crawler` `Web` `Code` `Data Pipeline` `Document Parsing` `Tool Use` `Skill`
